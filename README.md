## 📖 Sobre o Projeto

O **osint-boy** é um utilitário de reconhecimento em fontes abertas (OSINT) leve e modular, projetado para facilitar a coleta de dados e a análise de superfície de ataque em investigações digitais. 

Inspirado na estética *wasteland*, ele foca em ser direto, eficiente e fácil de integrar em pipelines de segurança.

## 🚀 Funcionalidades Principais

* **Subdomain Enumeration**: Mapeamento rápido de ativos.
* **Geo-IP Lookup**: Localização aproximada de alvos de rede.
* **Email Scraper**: Identificação de pontos de contato em domínios.
* **Social Check**: Busca automatizada por perfis em diversas plataformas.

## ⚙️ Instalação

```bash
# Clone o repositório
git clone [https://github.com/chariot27/osint-boy.git](https://github.com/chariot27/osint-boy.git)

# Entre na pasta
cd osint-boy

# Instale as dependências
pip install -r requirements.txt

#rodar
pip osintboy {domain}
```
